Curator

by Quentin Hill

Description

Curator is a static website builder written in PHP.

Always remember: Keep it simple.

Authors

  • Quentin Hill (quentin@quentinhill.com / quentinhill.com)

Inspiration

Curator is nothing new. There is a large selection of site generators out there.

Curator was created as a solution to mainting my personal website. It is unleashed unto the world under the MIT license.

Want to shop around a little more? Check out the following:

  • Phrozn: An amazing PHP-based site generator.

  • CakePHP: The opposite of a static site is a fully dynamic one.
